## üé≠ Animations & Interactions

### Global Animations

**@keyframes gradient-slide**
```css
0% ‚Üí 100%: Background position shift
Creates flowing gradient effect
Used in: Buttons, hero elements
```

**@keyframes pulse**
```css
0%, 100%: scale(1)
50%: scale(1.05)
Creates gentle breathing effect
```

**@keyframes float**
```css
Vertical translation: 0 ‚Üí -20px ‚Üí 0
Creates floating sensation
Duration: 3-4s infinite
```

**@keyframes fadeInUp**
```css
opacity: 0 ‚Üí 1
transform: translateY(30px) ‚Üí 0
Smooth entry animation
```

**@keyframes rotate**
```css
transform: rotate(0deg) ‚Üí rotate(360deg)
Continuous circular motion
Duration: 20s linear infinite
```

**Card Hover Effects:**
- `transform: translateY(-10px)` - Elevation
